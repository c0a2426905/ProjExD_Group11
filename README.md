# こうかとんシューティング

 ![title](fig/screen_shot.png)

## 実行環境の必要条件
* python >= 3.13.2
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクターこうかとんを操作する弾幕シューティングゲーム
* 参考URL：[ゼビウス](https://www.arcadearchives.com/title/aca-175/)

## ゲームの遊び方
* 敵が撒き散らす大量の弾（弾幕）を回避しながら、敵を撃墜していく
* スペースキーでゲーム開始する・ビームを出す
* 矢印キーで移動する
* 敵の撃破(10ポイント)とポイントアイテム(50ポイント)の回収でスコアアップ
* ゲームオーバーで強制終了もしくは右上の×ボタンでゲーム終了する

## ゲームの実装
### 共通基本機能
* キャラクター操作と背景画像
* 敵

### 分担追加機能
* メニュー画面(瀬川): 起動時にメニュー画面を表示,SPACEキーでゲーム開始
* 背景画像スクロール機能(大谷木)：ゲーム中の背景は下に進む
* ライフ機能（担当：結城）：こうかとんがダメージを負った場合に残りライフを一つ減らし、ライフが0になったらゲームオーバーにする機能。
* スコア機能(布瀬) ：敵撃破時10ポイント、撃破時10％の確率で出現するアイテムを回収で50ポイント
* ボスキャラ機能(シン)：ボス攻撃パターン、弾を作る
