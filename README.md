# こうかとんゼビウスゲーム

## 実行環境の必要条件
* python >= 3.13.2
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクターこうかとんをキーボード操作により弾幕シューティングゲーム
* 参考URL：[東方Project]

## ゲームの遊び方
* 敵が撒き散らす大量の弾（弾幕）を回避しながら、敵を撃墜していく。

## ゲームの実装
### 共通基本機能
* キャラクター操作と背景画像
* 敵

### 分担追加機能
* メニュー画面(瀬川): 起動時にメニュー画面を表示,SPACEキーでゲーム開始
* 背景画像スクロール機能
* スコア機能
* ライフ機能（担当：結城）：こうかとんがダメージを負った場合に残りライフを一つ減らし、ライフが0になったらゲームオーバーにする機能。
* スコア機能(布瀬) 
：敵撃破時10ポイント、撃破時10％の確率で出現するアイテムを回収で50ポイント
* ライフ機能
* ボスキャラ機能

### メモ
* こうかとんゼビウスゲーム

* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
* 衝突判定に用いた変数はbird,bombs,Trueとしている
